# AI智能刷题平台

一个纯前端静态网页刷题工具，初衷是为了备考**华为 HCIA-AI 认证**考试，同时也方便分享给朋友使用。支持自定义题库导入、练习模式、模拟考试、错题复习等功能，所有数据保存在浏览器本地。

## 快速开始

1. 下载 `index.html` 和题库 JSON 文件
2. 用浏览器打开 `index.html`
3. 点击 **导入题库** 按钮，选择一个 JSON 题库文件即可开始刷题

> 导入的题库会自动缓存到浏览器本地存储，下次打开无需重新导入。

## 功能概览

### 首页仪表盘
- 展示当前题库总数、已刷题目数、正确率、错题待复习数
- 支持导入/导出题库和学习进度

### 练习模式
- **顺序刷题**：按题库顺序逐题练习
- **随机刷题**：随机打乱题目顺序
- **错题重刷**：专门复习做错的题目
- 选择答案后即时显示对错和详细解析
- 支持收藏题目、题号跳转

### 考试模式
- 60 分钟倒计时，40 道题目
- 总分 1000 分，600 分及格（模拟 HCIA-AI 考试标准）
- 答题卡面板，可快速跳转任意题目
- 支持标记题目以便回头检查
- 交卷后显示得分、通过状态和错题列表

### 数据管理
- 学习进度自动保存到浏览器本地
- 支持导出学习进度（JSON 格式），可跨设备迁移
- 自定义题库导入后自动缓存

## 题库格式

题库使用 JSON 数组格式，每道题包含以下字段：

```json
[
  {
    "id": 1,
    "type": "单选题",
    "q": "题目内容",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "ans": "A",
    "exp": "答案解析：包含翻译、选项说明、为什么选正确答案等"
  }
]
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | number | 题目序号 |
| `type` | string | 题目类型，如 `单选题`、`多选题` |
| `q` | string | 题目内容 |
| `options` | string[] | 选项列表 |
| `ans` | string | 正确答案（多选题用字符串如 `"AB"` 或数组） |
| `exp` | string | 答案解析 |

## 技术栈

- 纯 HTML/CSS/JavaScript，无框架依赖
- 单文件应用，无需构建工具
- 使用 `localStorage` 实现数据持久化
- 知识分类自动标签（AI概览、机器学习、深度学习、开发框架、华为AI平台等）

## 项目初衷

这个网站最初是为了备考**华为 HCIA-AI 认证**而开发的。HCIA-AI 考试包含人工智能基础、机器学习、深度学习、华为 AI 平台（昇腾/Ascend、MindSpore、CANN 等）等知识点，需要一个方便随时刷题的工具。

后来也分享给朋友使用，现在扩展支持自定义题库导入，任何选择题类型的题目都可以用它来练习——比如四级英语词汇训练。
